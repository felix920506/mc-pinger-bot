# mc-pinger-bot
Discord bot that returns ping values of servers specified in config

## commands

### ping
  usage: `ping [id]`
  
  if id is not specified then all servers will be listed

## usage

### requirements
  - discord.py: `pip install discord.py`
  - mcstatus: `pip install mcstatus`
  - python3 environment

### running
  `python3 main.py`
